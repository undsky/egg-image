# egg-image

> Egg plugin for image


## 安装

```bash
$ npm i egg-image --save
# or
$ yarn add egg-image
```

## 依赖说明

### 依赖的 egg 版本

egg 2.x | egg 1.x
--- | ---
😁 | ❌

### 依赖的插件

[gm](https://github.com/aheckmann/gm)

## 开启插件

```js
// {app_root}/config/plugin.js
exports.image = {
  enable: true,
  package: 'egg-image',
};
```

## 配置

```js
// {app_root}/config/config.default.js
config.image = {

};
```
